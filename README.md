Website Choice
Organisation Description:
I have decided to develop a website for my own organization, Fuse Techni NPC. It is a technology-based non-governmental organization (NGO).
First, we have to understand what a technology-based NGO is. It is a non-profit organization that uses technology to deliver its vision and mission efficiently. It alters traditional methods to make a faster, easier and efficient work environment.
Fuse Techni NPC is a technology-based NGO that is originally based and founded in South Africa KwaZulu Natal, Durban. It focuses on bridging the gap between South African citizens and technology. Its purpose is to help these citizens build up their skills in the new technologically advanced world. It does this by providing digital skills training as it focuses more on the youth of South Africa that is live in disadvantaged communities. It provides them with digital skills that will benefit them in the workforce.
They could grow and expand nationally, even internationally with a new and new improved website. It will help them market their work and allow for international partnership, thus creating job opportunities for South Africans abroad. 
Fuse Techni NPC is currently lacking a proper marketing platform as well as funding and recognition.

Reasons for Choosing the Organisation:
I chose this organization because it bridges the existing gap between South African youth living in disadvantaged communities. It is one of the missing links in South Africa’s economic industry. By exposing it globally, it will boost economic growth by hindering the unemployment rate, especially amongst the youth.
 
Website Goals and Objectives

Aim of the Website:
The aim is to help the technologically disadvantaged and unemployed youth in underprivileged communities. Using the website, they can reach out to Fuse Techni NPC enquire and apply with all the required documentation. From there, supplementary steps will be taken. This website will help Fuse Techni NPC expand, make it user-friendly, efficient, reliable and easier to function. It will assist the NGO with marketing and hopefully attract potential investors. Furthermore, it will help them with their marketing, funding and recognition challenges.

How Will the Website Benefit the Organisation?
This website will not only benefit Fuse Techni locally or regionally but also internationally. It will help market their work as they continue to produce digitally skilled young adults. It will also assist them with their operations and functionality. This will promote and advance their connections leading to stimulated success of the organization.
The website will be easy to operate for users and provide enough information about the organization. From its operations, vision and mission to recruitments and benefits from being with Fuse Techni NPC.

Current Website Analysis 
Current Website Analysis
The current Fuse Techni NPC website is a newly developed website intended to provide information about the organisation, its services, and opportunities available to youth. The website was analysed to identify strengths and areas for future improvement.

Strengths of the Current Website
• Provides clear information about the organisation's mission and vision.
• Includes dedicated pages for Home, About, Services, Support, Contact, and Donations.
• Easy navigation structure connecting all pages.
• Responsive design that supports mobile and desktop devices.
• Interactive elements such as service cards and FAQ sections.

Weaknesses and Areas for Improvement
• Limited online application functionality for programme enrolment.
• No user account or member login system.
• Limited multimedia content such as videos and success stories.
• Search Engine Optimisation can be improved further.
• Additional accessibility features can be implemented.
• Future integration of online learning resources and student progress tracking.

Steps of Hosting a website
Hosting the Website on GitHub Pages
Step 1: Creating a GitHub Account
The first step in hosting the Fuse Techni NPC website is creating a GitHub account. The account provides access to repositories, version control features, and GitHub Pages hosting services.
Step 2: Creating a Repository
After creating an account, a new repository is created on GitHub. The repository is configured as public to allow GitHub Pages to publish the website online.
Step 3: Organising Website Files
Before uploading the website, all project files are organised into a structured folder system. The main pages, stylesheets, scripts, and images are placed in their respective folders to ensure that links and file paths function correctly when the website is published.
Step 4: Uploading Files to GitHub
Once the files are organised, upload to the GitHub repository. GitHub will store the website files securely and track all changes made to the project. This makes it easy to update, manage, and maintain the website over time.
Step 5: Enabling GitHub Pages
GitHub then automatically generates a public website from the uploaded files and makes it accessible through a web address.
Step 6: Deploying the Website
After GitHub Pages is enabled, the platform builds and deploys the website. The deployment process converts the repository contents into a live website that can be viewed by users through a web browser.
Step 7: Accessing the Live Website
Once deployment is complete, GitHub provides a unique website URL. This URL can be shared with users, stakeholders, and project assessors to access the live version of the Fuse Techni NPC website from anywhere with an internet connection.
Step 8: Testing and Maintenance
Regular maintenance is performed by updating content, fixing issues, and uploading changes to the GitHub repository, which automatically updates the live website.
